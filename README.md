# myblog-admin

博客管理系统（v1.0）

## 简介
本项目是基于 Vue3 + Ant Design Vue 的博客管理后台，支持内容管理、用户管理、评论管理、标签与分类管理等功能。后端采用 Django + DRF。

## 主要功能
- 博客动态管理
- 分类与标签管理
- 用户与权限管理
- 评论管理
- 文件上传与管理
- 支持多角色与权限

## 技术栈
- 前端：Vue 3, Ant Design Vue, Pinia, Vue Router, Axios, Vite
- 后端：Django, Django REST framework
- 数据库：MySQL 8.0+

## 快速开始
```bash
git clone https://github.com/yourusername/myblog-admin.git
cd myblog-admin
npm install
npm run dev
```

后端请参考后端目录或文档。
